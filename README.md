### Exercícios - Bash Script 

## Exercício 1
```Bash
chmod +x relacao.sh or just use `source`
$ ./relacao.sh 3 5
3 é menor que 5
$./relacao.sh 5 3
5 é maior que 3
$ ./relacao.sh 5 5
5 é igual a 5
```
